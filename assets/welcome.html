Welcome to Elfeed2

This is a standalone feed reader, successor to Emacs Elfeed. The browser-style left/center/right pane layout: feeds list, entry list, this preview pane.

Configure your feeds

A starter config was just written for you with commented examples. Edit it in your text editor of choice, then run File → Reload Config (Ctrl+Shift+R) — Elfeed2 picks up changes without restarting. Paths:

  • macOS / Linux: ~/.config/elfeed2/config
  • Windows: %USERPROFILE%\.config\elfeed2\config

Already have a classic Elfeed database? File → Import Classic Elfeed Index… ingests its ~/.elfeed/index as a one-way conversion (existing tags and read state preserved).

Daily-use keys

j / k Move down / up
g / G Jump to first / last entry
Enter Open entry detail view
b Open in browser (also marks read)
y Copy entry link
r / u Mark read / unread
v Toggle visual selection
Ctrl-A Select all
s or / Edit filter
f Fetch all feeds
d Download enclosure
D Toggle Downloads panel
l Toggle Log panel

The filter bar above the entry list controls what's shown: +unread only unread, @1-week only past week, =acoup.blog only that feed, !sponsor exclude titles matching, etc. Double-clicking a feed in the left pane jumps to that source.
